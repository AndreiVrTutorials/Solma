## 🚀 Instalación y Configuración

Sigue estos sencillos pasos para dejar el proyecto funcionando en tu entorno local:

1. **Descargar el proyecto:**
   * Haz clic en el botón verde **Code** de este repositorio y selecciona **Download ZIP**.
   
2. **Descomprimir el archivo:**
   * Extrae el contenido del archivo `.zip` en tu ordenador.

3. **Preparar el servidor local (XAMPP):**
   * Dirígete a la ruta de tu servidor local (por defecto `C:\xampp\htdocs`).
   * Crea una nueva carpeta llamada **`Solma`**.

4. **Desplegar los archivos:**
   * Copia los 7 archivos extraídos del ZIP y pégalos dentro de la carpeta que acabas de crear:
     ```text
     C:\xampp\htdocs\Solma\
     ```

5. **Configurar la Base de Datos:**
   * Abre **phpMyAdmin** (o tu gestor SQL preferido).
   * Crea una nueva base de datos.
   * Importa y ejecuta las consultas del archivo `.sql` incluido en el proyecto.

6. **Verificar la conexión:**
   * Abre tu navegador web e ingresa a: `http://localhost/Solma/index.html`
   * Asegúrate de que no haya errores de conexión y que el proyecto cargue correctamente.
